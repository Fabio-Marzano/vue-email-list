PROBLEMA 
Attraverso l'apposita API di Boolean
https://flynn.boolean.careers/exercises/api/random/mail
generare 10 indirizzi email e stamparli in pagina all'interno di una lista.

SOLUZIONE
1- Creo un array per inserire l'email da generare e attivo la funzione di return.
2- Genero l'email all'interno del DOM 
3- Avvio un ciclo for per generare l'inserimento delle 10 emails random.
4- Utilizzo l'API di Boolean per mostrare l'email. 
5- Confronto il risultato dell'email create utilizzando Thunder Client. 
6- Pusho l'email all'interno del DOM.
7- Mostro il risultato finale.
